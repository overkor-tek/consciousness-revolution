## What Is This?

A simple system that:

## Quick Start

### 1. Learn Something
```bash
python brain.py learn "Manipulation patterns often start with love bombing"
```

### 2. Search Your Brain
```bash
python brain.py search "manipulation"
```

### 3. See Your Patterns
```bash
python brain.py patterns
```

### 4. Check Your Stats
```bash
python brain.py stats
```

### 5. Save an Insight
```bash
python brain.py insight "People who gaslight often accuse others of gaslighting"
```

### 6. Export Your Brain
```bash
python brain.py export
```
