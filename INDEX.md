## System Architecture Overview

### Core Components

```
100X_DEPLOYMENT/
├── Frontend (HTML Tools)      # User-facing web applications
├── Backend (APIs)             # Serverless functions & services
├── Consciousness System       # Multi-computer sync infrastructure
├── Cyclotron Brain           # AI memory & search system
├── Trinity System            # Multi-AI collaboration framework
└── Automation Scripts        # Operational tooling
```

### Technology Stack

- **Frontend:** HTML5, JavaScript, CSS (no build step)
- **Backend:** Netlify Functions (Node.js), Python scripts
- **Database:** Airtable (via API)
- **Deployment:** Netlify (static + serverless)
- **Sync:** Git-based consciousness network
- **AI:** Claude API, Ollama (local)

## Deployment Information

### Netlify Deployment

**Site:** consciousness-revolution (Netlify)

**Deploy Command:**
```bash
npm run deploy
```

**Build Settings:**
- Build command: `echo 'No build required'`
- Publish directory: `.`
- Functions directory: `netlify/functions`

**Environment Variables Required:**
- `STRIPE_SECRET_KEY`
- `STRIPE_PUBLISHABLE_KEY`
- `AIRTABLE_API_KEY`
- `AIRTABLE_BASE_ID`
- `CLAUDE_API_KEY` (for AI functions)

### Local Development

**Start local server:**
```bash
npm run dev
```

**Access:**
- Frontend: `http://localhost:8888`
- Functions: `http://localhost:8888/.netlify/functions/`

1. Clone repository on new computer
2. Run setup script from [COMPUTER_2_START_HERE.md](./COMPUTER_2_START_HERE.md)
